# somenoisy

A learning sandbox for **[Strudel](https://strudel.cc)** — the browser-based live-coding music environment, a JavaScript port of [TidalCycles](https://tidalcycles.org).

This repo is mostly notes and reading material while I get comfortable with the language. Patches will live alongside the docs as I write them.
